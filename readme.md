# YTDownloader CLI

YTdownloader CLI is a command line tool to make easy downloading videos from Youtube.
|params|example|type|
|-|-|-|
|-u, --url| Video URL| String
|-o, --output| Output Directory| String
|-h, --help| Help | None
|-v, --version| Version Information | None
